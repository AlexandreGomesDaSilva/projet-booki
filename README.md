# Booki — Page d'Accueil d'une Agence de Voyage

## Description

Intégration de la maquette d'une plateforme de planification de vacances permettant aux utilisateurs de rechercher des hébergements et des activités touristiques à Marseille. Le projet consistait à transformer une maquette Figma en site web responsive en HTML et CSS, en respectant les spécifications du designer.

Projet réalisé dans le cadre de la formation **Développeur Web** — OpenClassrooms.

## Compétences développées

- **Intégrer du contenu conformément à une maquette** — Transformation fidèle d'une maquette Figma en page web structurée avec HTML5 sémantique et CSS3
- **Implémenter une interface responsive** — Adaptation du site à trois formats d'écran (desktop, tablette, mobile) via les media queries et Flexbox

## Travail réalisé

### Intégration de la maquette
- Structure sémantique HTML5 (`header`, `main`, `nav`, `section`, `footer`)
- Mise en page complète avec **Flexbox**
- Barre de recherche, filtres (Économique, Familial, Romantique, Nos pépites)
- Galerie de 6 hébergements + 3 hébergements populaires avec système d'étoiles
- Section de 4 activités touristiques (Vieux-Port, Fort de Pomègues, Calanques, Notre-Dame-de-la-Garde)
- Footer structuré en trois colonnes

### Design responsive
| Format | Largeur | Disposition |
|--------|---------|-------------|
| Desktop | > 1024px | Hébergements (65%) + Populaires (32%) côte à côte |
| Tablette | 768px – 1023px | Sections empilées verticalement |
| Mobile | < 768px | Pleine largeur, populaires affichés en premier |

## Technologies

- **HTML5** — Balisage sémantique
- **CSS3** — Flexbox, media queries, responsive design
- **Font Awesome 6.2.1** — Icônes
- **Google Fonts** — Police Raleway

## Structure du projet

```
projet-booki/
├── index.html
├── css/
│   └── style.css
└── images/
    ├── hebergements/
    ├── activites/
    └── logo/
```

## Lancer le projet

Ouvrir `index.html` dans un navigateur.

# Booki — Travel Agency Homepage

## Description

Integration of a mockup for a vacation planning platform that allows users to search for accommodations and tourist activities in Marseille, France. The project involved converting a Figma mockup into a fully responsive website using HTML and CSS, following the designer's specifications.

Project completed as part of the **Web Developer** program — OpenClassrooms.

## Skills Developed

- **Integrate content according to a mockup** — Faithful conversion of a Figma mockup into a structured web page using semantic HTML5 and CSS3
- **Implement a responsive interface** — Adaptation of the site to three screen formats (desktop, tablet, mobile) using media queries and Flexbox

## Work Completed

### Mockup Integration
- Semantic HTML5 structure (`header`, `main`, `nav`, `section`, `footer`)
- Full layout built with **Flexbox**
- Search bar, filter buttons (Budget-friendly, Family, Romantic, Hidden gems)
- Gallery of 6 accommodations + 3 popular accommodations with star rating system
- Section featuring 4 tourist activities (Vieux-Port, Fort de Pomègues, Calanques, Notre-Dame-de-la-Garde)
- Three-column footer

### Responsive Design
| Format | Width | Layout |
|--------|-------|--------|
| Desktop | > 1024px | Accommodations (65%) + Popular (32%) side by side |
| Tablet | 768px – 1023px | Sections stacked vertically |
| Mobile | < 768px | Full width, popular section displayed first |

## Technologies

- **HTML5** — Semantic markup
- **CSS3** — Flexbox, media queries, responsive design
- **Font Awesome 6.2.1** — Icons
- **Google Fonts** — Raleway typeface

## Project Structure

│   └── style.css
└── images/
    ├── hebergements/
